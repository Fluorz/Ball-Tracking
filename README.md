# Ball Tracking

[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://travis-ci.org/joemccann/dillinger)

# New Features!

  - Detect movement
  - Color picker

### Usage

Install the dependencies with pip

Motion detector
```sh
$ python motion_detector.py --video [VIDEO]
$ python motion_detetcor.py
```

Color picker

```sh
$ python color_picker.py -i [IMAGE]
```
